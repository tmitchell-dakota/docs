## How to Use

1. **Copy** any of the JSON examples below.
2. **Go to the API playground** by clicking on the **"Try it"** button.
3. **Paste** the JSON into the request body field.
4. **Click "Send"** button to execute the request.

## Examples

### Count Investment Records
```json
{
  "module": "investment__c",
  "count_only": "1"
}
```

### List All Investments
```json
{
  "module": "investment__c"
}
```

### List Investments with Specific Fields
```json
{
  "module": "investment__c",
  "filters": {
    "fields": ["name", "createddate", "country__c"],
    "order_by": "name:DESC",
    "max_num": "-1"
  }
}
```

### Filter By Investment Name (AND Example)
```json
{
  "module": "investment__c",
  "filters": {
    "filter": [
      {
        "$and": [
          {
            "name": {
              "$contains": "RT"
            }
          },
          {
            "name": {
              "$contains": "Test"
            }
          }
        ]
      }
    ]
  }
}
```

### Filter By Created Date (Last 30 Days)
```json
{
  "module": "investment__c",
  "filters": {
    "filter": [
      {
        "createddate": {
          "$dateRange": "last_30_days"
        }
      }
    ],
    "max_num": "-1"
  }
}
```

### Filter By Fund Balance
```json
{
  "module": "investment__c",
  "filters": {
    "filter": [
      {
        "fund_balance__c": {
          "$gt": "1000000"
        }
      }
    ]
  }
}
```
